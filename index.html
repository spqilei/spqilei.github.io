TS的类型声明

1、基本类型

  • 类型声明
    • 类型声明是TS非常重要的一个特点
    • 通过类型声明可以指定TS中变量（参数，形参）的类型
    • 指定类型后，当为变量赋值时，TS编译器会自动检查是否符合类型声明，符合则赋值，否则报错
    • 语法
  • 自动类型判断
    • TS拥有自动的类型判断机制
    • 当对变量的声明和赋值时同时进行的，TS编译器会自动判断变量的类型
  • 类型：
    类型 例子 描述
    number 1，-6,2.8 任意数字
    string ‘hi’,’123’ 任意字符串
    boolean true,false 布尔值true或false
    any * 任意类型
    unknown * 类型安全的any
    void 空值（undefined） 没有值（或undefined）
    never 没有值 不能是任何值
    object {name：’熏悟空’} 任意js对象
    array [1,2,3] 任意js数组
    tuple [4,5] 元组，TS新增类型，固定长度数据
    enum enum{A,B} 枚举，TS中新增类型

object 对象

array 数组

tuple 元祖

enum 枚举
